# Krew

Krew is a WinUI 3 Windows desktop app designed to help you organise and launch your game library. This app focuses on simplicity and usability.



## Walkthrough

1. **Add Games Manually:**  
   You can start by adding your games along with their metadata (such as name, cover image URL, and tags) directly into the app's database.

2. **Browse Your Library:**  
   Navigate to the **Library** page to see all your games. Use the search box to filter games by name or tags.

3. **Pin Your Favourites:**  
   Pin games you frequently play for faster access. Pinned games appear in the navigation pane.

4. **Launch Games:**  
   Select any game from the library or pinned list to launch it. You will be prompted with a confirmation dialogue before launching.

5. **Manage Pinned Games:**  
   Right-click on pinned games to quickly launch or unpin them via the context menu.

6. **Access Settings:**  
   Open the **Settings** page to adjust app preferences like sound (on/off). The app follows the default Windows themes.



## Features

- Manually add games with metadata  
- Browse and search your game library  
- Pin games for quick access  
- Confirmation dialogue before launching games  
- Context menu for pinned games (launch/unpin)
